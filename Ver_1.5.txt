2025/11/6 
V1.0 
 
This version mainly uses motion files to complete the robot's acquisition of the ball's position and completes kicking by moving forward, left, and right 
 
The main issue is 
1. Lack of stability makes it easy to fall when playing left or right turns while kicking a ball. It is necessary to modify the data inside the motion file 
2. The function of standing up after falling has not been completed, and the data inside the motion file needs to be modified 
3. Inaccurate state judgment, continuous judgment should not be made during the action process, and logic needs to be improved 
4. The logic for determining the location of the best hitting point needs to be more accurate and improved 
5. Parameters need to be reset and optimized 
6. The angle set in motion does not match the limit of Nao6, exceeding the joint limit of the robot. This may be the main reason affecting 1.2 and needs to be confirmed
 
Task to be completed 
1. Update the codes for the defender (emergency) and goalkeeper 
2. Increase rotation to allow the robot to automatically adjust direction 
3. Update the forward motion to allow the robot to select a more suitable motion file when it is far away from the ball
